# Snackbars

[Snackbars](https://material.io/guidelines/components/snackbars-toasts.html) 通过消息提供简短的反馈——通常是在屏幕的底部。

snackbar包含与执行的操作直接相关的一行单一文本。
它们可能包含文本操作，但没有图标。

一次只能显示一个snackbar。
