1. React-Bootstrap

由React组件构建的Bootstrap 3组件

在积极开发- api将会改变. 查看 1.0.0 路线图 和 贡献参考 看看你能帮上什么. 在1.0.0版本之前, 取消或中断更改将导致一个小的版本的冲突.

1.1. 文档

请参阅带有实时可编辑示例英文原版文档、中文文档

1.2. 相关模块

  • react-router-bootstrap – 集成 React Router
  • React Bootstrap 扩展 - React Bootstrap的一个版本，其中每个组件都有一些额外的props，它们映射到bootstrap的实用程序类 比如, pullRight添加类 pull-right 到一个组件.

1.3. 本地设置

  • 使用 npm install安装依赖关系
  • 使用npm test运行测试
  • 使用npm run tdd来在观察模式下运行测试
  • 使用npm run docs来在开发者模式下运行文档. 当您工作时，文件的变更时可以被看到。只需刷新页面，就可以看到更新
  • 使用npm run build进行构建
