1. Material-UI@v1-beta

Material-UI 是一组 React 组件，它实现了 Google's Material Design 规范.

1.1. 安装

Material-UI 是一个可用的 npm package.

稳定的渠道

npm install material-ui

预发布渠道

npm install material-ui@next

请注意，@next只会指向预发布;要获得最新的稳定版本，请使用@latest代替。

最新稳定版本

npm install material-ui@latest

1.2. 用法

这里有一个快速开始的例子，这是你需要的:

import React from 'react';
import { render } from 'react-dom';
import Button from 'material-ui/Button';

function App() {
  return (
    <Button>
      Hello World
    </Button>
  );
}

render(<App />, document.querySelector('#app'));
